A Jobs searching platform built using MERN Stack.

Technologies used:

1. ReactJS

2. NodeJS

3. MongoDB

4. ExpressJS

5. Cloudinary (for storing files and images)

6. Socket.io (for chat feature)

7. Stripe (for payment gateway)

Description:

A job searching platform for climate tech companies where companies can post jobs and professionals can apply for them. Payment gateway and chat feature are two worthy mentions from the above project.
